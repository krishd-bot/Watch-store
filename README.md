# ⌚ Watch Store

A full-stack MERN Watch Store application built for watch enthusiasts and collectors. This project allows users to explore premium watches, browse collections by category, and experience a modern e-commerce workflow.

## 🚀 About The Project

This project was created out of a personal interest in watches and modern web development. The goal is to build a scalable watch marketplace where users can discover stylish and luxury watches through a clean and responsive interface.

Currently, the application focuses on displaying watch collections and category-based browsing. Future updates will transform it into a complete e-commerce platform.

---

* Node.js
